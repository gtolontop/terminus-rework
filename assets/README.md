# Assets

Ce dossier sert de reserve d'assets pour le rework.

## Dossiers

- `original-terminus/img/` : images du jeu HTML original.
- `terminus-rpg/sprites/` : sprites du prototype Godot de rework.

## Regle de travail

Les assets bruts restent ici. Quand on commencera a coder la premiere partie, on pourra creer des dossiers de production plus stricts, par exemple `characters/`, `tilesets/`, `ui/` et `locations/`, sans perdre la trace des sources.
